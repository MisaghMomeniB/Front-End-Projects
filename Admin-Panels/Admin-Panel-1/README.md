## **📦 Installation**
### **1️⃣ Clone the Repository**
```bash
git clone https://github.com/your-username/admin-dashboard.git
cd admin-dashboard
```

### **2️⃣ Install Dependencies**
```bash
npm install
```

### **3️⃣ Start the Development Server**
```bash
npm run dev
```
The app will run at: **http://localhost:5173/**
